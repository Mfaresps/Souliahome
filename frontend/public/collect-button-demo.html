🚀 Optimization Notes

Animations use hardware acceleration (GPU). Mobile animations are automatically disabled for better performance. No JavaScript polling or timers are used—pure CSS animations.

Quick Implementation

Step 1: Add to HTML Head

<link rel="stylesheet" href="/collect-highlight.css"> <script defer src="/collect-highlight.js"></script>

Step 2: Add Class to Button

<button class="btn btn-primary btn-collect" onclick="doCollectMovement(...)"> تحصيل </button>

Step 3: Done!

The button will automatically have all the highlight effects. No additional code needed!

Optional JavaScript Integration
// Flash button on new collection const btn = CollectHighlight.getByTransactionId('tx-123'); CollectHighlight.flashButton(btn); // Add pending badge CollectHighlight.addPendingBadge(btn); // Show loading state CollectHighlight.setLoading(btn, true); // Show success CollectHighlight.showSuccess(btn);
