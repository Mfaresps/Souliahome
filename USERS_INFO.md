## 🔧 الخطوات لتطبيق البيانات الجديدة:

### 1️⃣ حذف قاعدة البيانات الحالية
```bash
cd backend
# احذف ملف MongoDB (إن وجد)
# أو قم بتشغيل: npm run seed:reset
```

### 2️⃣ تشغيل البإعادة إنشاء البذر
```bash
npm run start
```

> البذر سيتم تطبيقه تلقائياً عند بدء التطبيق

### 3️⃣ اختبر المستخدمين في واجهة المستخدم
- ادخل مع `admin` / `Fares@2024`
- الخزنة محمية بكلمة مرور: سأخبرك بها
