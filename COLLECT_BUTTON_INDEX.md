## 🔧 Technical Resources

### CSS File Details
**Location:** `/frontend/public/collect-highlight.css` (12 KB)

**What it contains:**
- Base button styling (gradient, shadows)
- Glow animation (2.5s loop)
- Shadow pulse animation (3s loop)
- Shimmer effect (3s loop)
- Hover interactions
- Responsive breakpoints
- Dark mode variants
- State animations (loading, success)
- Badge indicators
- Accessibility features

**Key animations:**
- `collectGlow` - Border gradient pulse
- `collectShadowPulse` - Expanding glow
- `collectShimmer` - Light reflection
- `collectHighlightFlash` - Attention getter
- `badgePulse` - Pending indicator

### JavaScript File Details
**Location:** `/frontend/public/collect-highlight.js` (8 KB)

**What it contains:**
- `CollectHighlight.init()` - Initialize effects
- `CollectHighlight.applyCollectClasses()` - Auto-class application
- `CollectHighlight.setupDropdownHighlight()` - Dropdown effects
- `CollectHighlight.flashButton()` - Flash animation
- `CollectHighlight.addPendingBadge()` - Badge management
- `CollectHighlight.setLoading()` - Loading state
- `CollectHighlight.showSuccess()` - Success animation
- State management utilities

### Demo Page Details
**Location:** `/frontend/public/collect-button-demo.html` (15 KB)

**Features:**
- Interactive button demonstrations
- Dark/light mode toggle
- All effect states visible
- Responsive showcase
- Code examples
- Performance metrics
